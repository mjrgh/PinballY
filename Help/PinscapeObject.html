pinscape Object

pinscape is a pre-defined PinballY object that provides access to Pinscape Controller functions. The object represents the overall Pinscape software interface, covering any number of controller units installed on the system, rather than being tied to a particular unit.

Event target

pinscape is an event target, so you can use the standard event methods (on(), off(), addEventListener(), etc) to add and remove event listeners. See EventTarget.

The object doesn't currently define any events, but it might in the future.

Methods and properties

The pinscape defines the following methods and properties.

pinscape.nightMode: This property lets you determine the current Night Mode status on the Pinscape Controller device(s), and lets you change the status.

When read, the property returns true if any Pinscape units are currently in Night Mode, false if there are no controllers attached, or if none of them are in night mode. Note that the property reads as true if any controllers are in Night Mode, even if there are multiple controller present and they're not all in Night Mode. In most cases, the mode is turned on or off across all controllers, so it's unusual for the status to be mixed in practice, but if it is, the result is always true if even one unit has Night Mode enabled.

When written, the property sets Night Mode status on all attached controllers to the new value. Set the property to true to activate Night Mode on all controllers, and set it to false to return all controllers to normal "day mode" status.
